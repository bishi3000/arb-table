arb-table
=========

Bitcoin arbitrage table. Shows differences in exchanges rates of various bitcoin exchanges

Requires:

  Maven 3
  Java 7
  
  
This project uses Vert.x and the Vert.x maven plugin to run


To build:

mvn clean package

To run:

mvn vertx:runMod


Once running, go to :

http://localhost:8080/index.html
